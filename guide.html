Introduction

Cet ordinateur utilise une distribution de Linux appelée Debian 12. Une distribution est la combinaison de GNU/Linux et d'un ensemble de programmes préinstallés.

Parmi les programmes préinstallés sur Debian, on trouve notamment :

  • GNOME : l'interface utilisateur qui gère le bureau, les fenêtres, etc.
    GNOME : une empreinte de pied sur fond bleu.
  • Fichiers : le gestionnaire de fichiers, équivalent de l'« Explorateur de fichiers » sous Windows.
    Fichiers : une armoire à dossiers bleue.
  • Firefox ESR : navigateur internet.
    ESR signifie que c'est une version plus stable avec un support étendu.
    Firefox ESR : un renard orange entourant un globe.
  • Logiciels : le magasin d'applications pour télécharger la plupart des programmes.
    Logiciels : un sac de courses avec des formes géométriques colorées.
  • LibreOffice : l'équivalent de la suite Microsoft Office.
    LibreOffice : une feuille blanche avec un coin replié.
  • Plein de petits jeux sont préinstallés. On évite l'espionnage de Windows, pas les petits jeux pourris.

La particularité de Debian est sa grande utilisation de logiciels non commerciaux et sa très grande stabilité.

Interface GNOME

L'interface GNOME est ce que vous voyez à l'écran : le bureau, les fenêtres, les icônes, etc. Elle vous permet d'interagir avec l'ordinateur en utilisant la souris et le clavier.

Tableau de bord

Le tableau de bord, la barre en haut de l'écran, affiche le menu des applications, l'heure, le volume, le réseau, etc. C'est l'équivalent de la barre des tâches sous Windows.

En cliquant sur l'heure, un menu s'affiche avec les notifications, comme les informations sur les mises à jour installées.

Menu des applications

Cliquez sur Activités en haut à gauche ou appuyez sur la touche Super (la touche avec le logo Windows) pour entrer dans la vue Activités. Vous pouvez aussi glisser trois doigts vers le haut sur le pavé tactile.

Dans cette vue, vous verrez :

  • La barre de recherche : en tapant, vous recherchez dans la liste des programmes et de vos fichiers.
  • Votre bureau et les applications ouvertes : cliquez dessus pour afficher l'application.
  • Le dock : avec les raccourcis vers les applications les plus utilisées et les applications ouvertes.

En cliquant sur le carré avec neuf points dans le dock (ou en double-appuyant sur Super depuis une fenêtre normale), vous entrez dans la vue des Applications.

Cette fois, à la place des applications ouvertes, sont affichés :

  • Les bureaux créés. GNOME permet de créer plusieurs bureaux.
    Attention à ne pas naviguer de l'un à l'autre par inadvertance.
  • La liste des applications installées sur le système.

Ouvrir les applications et basculer d'une à l'autre

Pour ouvrir une application, cliquez sur son icône. Pour la fermer, comme sur Windows, cliquez sur la croix en haut à droite.

Pour passer entre les applications ouvertes, vous pouvez :

  • Utiliser la vue Activités et cliquer sur une fenêtre ouverte.
  • Utiliser le raccourci clavier Alt + Tab.

Le système de fichiers

En ouvrant l'application Fichiers, vous accédez au gestionnaire de fichiers dans le dossier Dossier personnel (Home en anglais).
Tous vos fichiers et dossiers doivent être créés dans Dossier personnel. Pour le reste, cela fonctionne comme sur Windows.
Comme sur Windows, des sous-dossiers Images, Documents, Vidéos, etc., sont pré-créés.

Naviguer sur Internet

Utiliser le navigateur par défaut : LibreWolf

Pour accéder à Internet, utilisez LibreWolf, un navigateur axé sur la sécurité et la confidentialité. LibreWolf est basé sur Firefox mais configuré avec des options renforçant la confidentialité.

LibreWolf : un loup sur un fond bleu.

Alternative : Firefox

Si un site web ne fonctionne pas correctement avec LibreWolf (cela peut arriver à cause de ses paramètres de sécurité stricts), utilisez Firefox.

Firefox : un renard orange entourant un globe.

Bureautique

L'équivalent de Microsoft Office sur Debian est LibreOffice, qui offre tous les programmes similaires.

LibreOffice : une feuille blanche avec un coin replié.

Applications LibreOffice

  • LibreOffice Writer : comme Word, pour les documents texte.
    LibreOffice Writer : une feuille avec un coin replié et un stylo bleu.
  • LibreOffice Calc : comme Excel, pour les feuilles de calcul.
    LibreOffice Calc : une feuille avec un coin replié et un tableau vert.
  • LibreOffice Impress : comme PowerPoint, pour les présentations.
    LibreOffice Impress : une feuille avec un coin replié et un écran orange.

Mettre à jour le système et les programmes

Il est important de garder l'ordinateur à jour pour des raisons de sécurité. Les mises à jour corrigent des bugs et des failles exploitables par des virus.
Sous Linux, les mises à jour ne sont pas gênantes et ne modifient pas vos réglages comme sur Windows.
Pour être fiable, l'ordinateur doit être à jour.

Mises à jour automatiques

Par défaut, les mises à jour sont installées automatiquement.
Lorsque vous éteignez l'ordinateur et que des mises à jour sont disponibles, il vous proposera de les installer (une petite case cochée dans le pop-up). Laissez la case cochée, ou cochez-la si elle ne l'est pas, et l'ordinateur s'occupera des mises à jour.
Lors de l'extinction, il se peut que l'ordinateur vous demande le mot de passe.

Note : Puisque les mises à jour sont automatiques, il n'est pas nécessaire de s'en inquiéter. Les points suivants sont pour information.

Vérifier manuellement avec "Logiciels"

  • Ouvrez le menu des applications en cliquant sur Activités.
  • Cliquez sur Logiciels.
    Logiciels : un sac de courses avec des formes géométriques colorées.
  • Allez dans l'onglet Mises à jour.
  • Si des mises à jour sont disponibles, cliquez sur Installer les mises à jour.

Utiliser le Terminal pour les mises à jour

Les opérations en ligne de commande se font avec l'application Terminal. Une fois ouverte, il suffit d'y taper les commandes.

Terminal : un écran noir avec un curseur blanc.
Astuce : Pour copier du texte depuis le Terminal, utilisez Ctrl + Shift + C au lieu de Ctrl + C.
Pour coller du texte, utilisez Ctrl + Shift + V.

Mettre à jour avec le Terminal

Dans le Terminal, tapez la commande suivante et appuyez sur Entrée : 

sudo apt update && sudo apt upgrade

Explications :

  • sudo : Exécute la commande avec les privilèges administrateur.
  • apt : Gestionnaire de paquets qui permet d'installer, de mettre à jour et de supprimer des programmes.
  • update : Met à jour la liste des programmes disponibles.
  • upgrade : Installe les mises à jour des programmes.
  • && : Enchaîne deux commandes l'une après l'autre.

La première fois après avoir tapé la commande sudo, l'ordinateur vous demandera de taper votre mot de passe. Ensuite, il le retient pour cinq minutes.

Installer un programme

Il existe plusieurs méthodes pour installer un programme sur Debian, classées ici par ordre de préférence.

Utiliser le magasin d'applications ("Logiciels")

La méthode la plus simple est d'utiliser le programme Logiciels :

  • Ouvrez Logiciels depuis le menu des applications.
    Logiciels : un sac de courses avec des formes géométriques colorées.
  • Utilisez la barre de recherche (en haut à droite) pour trouver l'application souhaitée.
  • Cliquez sur l'application dans la liste des résultats.
  • Cliquez sur le bouton Installer.

Exemple : Pour installer le lecteur vidéo VLC :

  • Recherchez "VLC" dans Logiciels.
  • Sélectionnez-le dans les résultats.
  • Cliquez sur Installer.

Installer via le Terminal avec apt

Si vous ne trouvez pas l'application dans Logiciels, utilisez le Terminal :

Mettre à jour la liste des programmes

Dans le Terminal, tapez la commande suivante :

sudo apt update

Explications :

  • sudo : Exécute la commande avec les privilèges administrateur.
  • apt : C'est l'application qui gère l'installation des programmes.
  • update : Met à jour la liste des programmes disponibles.

Rechercher une application

Dans le Terminal, tapez la commande suivante pour chercher une application : 

apt search nom_de_l_application

Explications :

  • apt : Le gestionnaire de paquets.
  • search : Commande pour rechercher des programmes.
  • nom_de_l_application : Remplacez par le nom du programme que vous cherchez.

Exemple : Pour chercher le lecteur vidéo VLC : 

apt search vlc

Une liste de toutes les applications correspondant à votre recherche s'affiche. Il faut trouver votre application dans la liste.

Installer une application

Une fois que vous avez trouvé le nom exact de l'application, installez-la avec cette commande : 

sudo apt install nom_de_l_application

Explications :

  • sudo : Exécute la commande avec les privilèges administrateur.
  • apt install : Commande pour installer une application.
  • nom_de_l_application : Remplacez par le nom du programme que vous voulez installer.

Exemple :

sudo apt install vlc

Installer en téléchargeant le fichier

Si l'application n'est pas disponible via Logiciels ou apt, téléchargez-la depuis le site officiel :

Attention : Utilisez cette méthode en dernier recours. Téléchargez uniquement depuis des sites officiels pour éviter les risques de virus ou de logiciels malveillants.
  • Rendez-vous sur le site officiel de l'application et téléchargez le fichier d'installation (généralement avec une extension .deb).
  • Ouvrez le dossier où le fichier a été téléchargé (souvent le dossier "Téléchargements").
  • Double-cliquez sur le fichier téléchargé.
  • Une fenêtre s'ouvre pour installer l'application. Cliquez sur Installer et suivez les instructions.
